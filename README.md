- o que precisar para roda o projeto ?

1- executar o comando npm install para instalar as dependencias:

    "cookie-parser": "~1.4.4",
    "debug": "~2.6.9",
    "ejs": "~2.6.1",
    "express": "~4.16.1",
    "http-errors": "~1.6.3",
    "morgan": "~1.9.1",

1- roda npm run dev no terminal 

de default ,  o servidor utilizar a porta 3000
